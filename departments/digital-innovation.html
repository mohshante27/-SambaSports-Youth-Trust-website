Mohamed is a passionate technologist dedicated to bridging the digital divide in Kwale County. He leads digital literacy programs and innovation initiatives, helping community members gain essential tech skills for the modern world.

✦ WHAT WE TEACH ✦

Digital Literacy & Innovation Skills

Empowering the community with essential tech skills for the future

Coding

Learn programming languages like HTML, CSS, JavaScript, Python, and more. Build websites and applications.

Robotics

Introduction to robotics, building and programming robots. Hands-on learning with Arduino and sensors.

Animation

2D and 3D animation techniques. Create engaging animated content using industry-standard tools.

Graphic Design

Learn design principles, Photoshop, Illustrator, and Canva. Create stunning visuals for any purpose.
